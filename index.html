Mapas
del mundo

Los mapas son experiencias interactivas y gratificantes, y no solo imágenes bonitas. Los mapas más valiosos son productos de información que son visualmente interesantes desde la primera vez que los ve. Sin embargo, lo recompensan con información adicional a medida que explora e interactúa con el mapa. Cuando toca el mapa, responde brindándole detalles sobre las cosas que toca: toque una tienda y le indicará las ventas de este año hasta la fecha con un gráfico de las ventas de los tres años anteriores.

Analisis de Datos

La analítica geoespacial se utiliza para agregar tiempo y ubicación a los tipos tradicionales de datos y para crear visualizaciones de datos. Estas visualizaciones pueden incluir mapas, gráficos, estadísticas y cartogramas que muestran cambios históricos y cambios actuales. En Aragon Map somos especialista en Analítica de datos.

Mapas Web

Un servicio de mapas facilita los mapas, las entidades y los datos de atributos disponibles dentro de muchos tipos de aplicaciones de cliente y puede exponer diferentes niveles de capacidades. En Aragon Map somos especialista en publicación de web maps y apps maps.

Proyectos

Mapas Outdoors

El mapa excursionista es la herramienta de orientación básica, tanto para senderistas como para montañeros. Un mapa es una representación en dos dimensiones de una porción de territorio. Para la seguridad en montaña los mapas que nos interesan son los mapas topográficos con detalle excursionista y de una escala 1:25.000 o 1:40.000.

Gestion Información

Proyecto de gestion de información en la construcción y gestion de autopistas. El usuario puede coordinar recolección de datos de terreno, evaluar el avance de obras y gestionar los permisos ambientales ante la autoridad competente.Una excelente herramienta para la gestion de activos.

Mapa Web

El proyecto consiste en la elaboración de mapas web de la ciudad de Valdivia y alrededores. En el mapa web el usuario puede interactuar desde un un pc o apps. El usuario puede buscar información, ejecutar navageción en vehiculo, etc.Herramientaideal para todo tipo de usuario.
